# Blog App

A full-stack blog application where users can create, edit, delete, and read blog posts.

## Features

- User Registration & Login
- JWT Authentication
- Create Blog Posts
- Edit Existing Blogs
- Delete Blogs
- View All Blogs
- Responsive User Interface
- Protected Routes


## Tech Stack

   ### Frontend
   - React
   - Vite
   - CSS / Tailwind CSS

   ### Backend
   - Node.js
   - Express.js

   ### Database
   - MongoDB
